PDF 양면 스캔 합치기 · 용량 줄이기

단면 스캐너로 따로 스캔한 앞면·뒷면 PDF를 원래 순서로 합치고, 여러 PDF를 하나로 모으고, 무거운 스캔본의 용량을 줄입니다. 파일은 서버로 업로드되지 않고 브라우저 안에서 처리되며, 무료이고 설치·회원가입이 필요 없습니다.

🔒 모든 처리는 당신의 브라우저 안에서만 이뤄집니다. PDF가 외부 서버로 전송되지 않습니다.

스캔 용량 줄이기
문제집·교재 스캔본이 너무 무거워 메일 첨부나 업로드가 안 될 때, 화질은 지키며 용량만 줄입니다.

여러 PDF 합치기
흩어진 PDF를 드래그로 순서 바꿔 하나의 파일로 모읍니다. 페이지 제외·회전도 가능합니다.

02사용법

  1. 도구 선택양면 스캔 합치기 · 순서대로 합치기 · 용량 줄이기 중에서 고릅니다.
  2. PDF 올리기업로드 칸을 클릭하거나 파일을 드래그합니다.
  3. (선택) 스캔 보정 켜기기울기·색상·용량을 보정합니다. 자동 모드는 색과 음영을 살리며 용량을 줄입니다.
  4. 실행 후 다운로드대표 페이지 미리보기로 확인한 뒤 전체에 적용하고 내려받습니다.

03자주 묻는 질문

단면 스캐너로 양면 문서를 어떻게 합치나요?
앞면만 모아 한 번, 뒷면만 모아 한 번 스캔한 뒤 두 PDF를 올리면 1·2·3·4… 순서로 자동 재조합됩니다. 뒷면을 뒤집어 스캔한 경우 역방향 옵션을 선택하세요.
파일이 서버로 업로드되나요?
아니요. 모든 처리는 브라우저(기기) 안에서 이뤄지며, PDF가 외부 서버로 전송되지 않습니다. 인터넷 연결은 첫 로딩에만 필요합니다.
무료인가요? 회원가입이 필요한가요?
완전히 무료이며 설치나 회원가입이 필요 없습니다. 페이지를 열면 바로 사용할 수 있습니다.
용량이 얼마나 줄어드나요?
스캔 품질과 설정에 따라 다르지만, 스캔 PDF는 보통 절반 정도로 줄어듭니다. 해상도(DPI)와 색상 모드를 조절해 화질과 용량의 균형을 맞출 수 있습니다.
용량을 줄이면 화질이 깨지나요?
자동 모드는 색과 음영을 유지하면서 용량만 줄입니다. 가장 작게 만드는 흑백 모드는 글자만 남기므로 도표나 음영은 사라질 수 있어, 글자 위주 문서에 적합합니다.
